Mood-Based Music Recommendation System

Overview
The Mood-Based Music Recommendation System is a Python project that suggests songs based on the user’s current mood.

By selecting a mood such as Happy, Sad, Energetic, Relaxed, or Romantic, the system analyzes a dataset of songs and recommends tracks that match the selected emotion.

This project demonstrates how data processing and simple recommendation logic can create personalized user experiences.

---

Features

 Mood-based song suggestions
 Simple and user-friendly interface (terminal based)
 Uses a real song dataset stored in CSV format
 Fast filtering and recommendation
 Easy to expand with more moods and songs

How It Works

 The program asks the user to enter their current mood
 The system searches the songs dataset
 Songs matching the selected mood are filtered
 Recommended songs are displayed to the user


How to Run the Project

Step 1

Clone the repository

git clone https://github.com/your-username/Mood-Based-Music-Recommendation-System.git

Step 2

Navigate to the project folder

cd Mood-Based-Music-Recommendation-System

Step 3

Run the Python program

python "Mood Based Music Recommendation System.py"
